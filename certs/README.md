# 本地证书目录

如果代理服务器使用自签名 TLS 证书，请把维护者提供的公开证书复制到这里，并命名为：

```text
mp-proxy.crt
```

随后在项目根目录设置：

```text
MP_PROXY_CA_CERT=./certs/mp-proxy.crt
```

证书文件会被仓库的 `*.crt` 规则忽略，不会随普通 Git 提交上传。

注意：

- 可以放置用于校验服务器身份的公开 `.crt` 证书；
- 不要放置服务器私钥、Basic Auth 密码或微信公众号 AppSecret；
- 任何 `.key` 私钥都不应分发给客户端用户；
- 从项目根目录以外的位置运行时，应改用证书绝对路径。
